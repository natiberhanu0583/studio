# **App Name**: CafeConnect

## Core Features:

- Customer Order Placement (Web/Android): Customers can browse the menu, select items, and place orders through a web or Android app interface.
- Waiter Order Management: Waiters receive new order notifications, view order details, and update the order status (e.g., 'Sent to Chef', 'Ready', 'Delivered').
- Chef Order Preparation View: Chefs can view incoming orders and manage the food preparation process using a simplified view.
- Notification system tool: Generative AI reasons about whether to send push notifications to waiters and customers for order updates (e.g., new order received, order status changes).
- Admin Income/Expense Export: Admin has access to an aggregate spreadsheet export containing details of cafe income/expense for record keeping and further manipulation in standard spreadsheet apps.
- Admin: User Management: Administrators can manage user accounts (waiters and chefs).

## Style Guidelines:

- Primary color: Warm terracotta (#E07A5F) evoking a cozy, inviting cafe atmosphere.
- Background color: Light beige (#F8F0E3), creating a soft, neutral backdrop.
- Accent color: Muted olive green (#81B29A), providing a natural, calming contrast.
- Headline font: 'Playfair' serif for an elegant, high-end feel; Body font: 'PT Sans' sans-serif to ensure readability
- Simple, line-based icons representing menu categories and order actions.
- Clean, card-based layout for menu items and order summaries, promoting ease of use.
- Subtle transitions and loading animations to enhance user experience without being distracting.
